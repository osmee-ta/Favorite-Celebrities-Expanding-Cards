# Favorite Celebrities – Expanding Cards

## Overview

**Favorite Celebrities – Expanding Cards** is a front-end web project built using **HTML**, **CSS**, and **JavaScript**. The project presents a collection of celebrity cards that expand smoothly when clicked, creating an attractive and interactive user experience.

The webpage features an animated title, responsive expanding cards, smooth transitions, and a professional footer.

---

## Features

* Interactive expanding cards
* Smooth CSS transitions and animations
* Animated page title with glow effect
* Responsive layout
* Celebrity names appear on the active card
* Local image support
* Professional footer with developer credit
* Simple and clean user interface

## How It Works

1. The webpage loads with the first celebrity card expanded.
2. Clicking on another card removes the active state from the previous card.
3. The selected card expands smoothly.
4. The celebrity's name fades into view using CSS transitions.
